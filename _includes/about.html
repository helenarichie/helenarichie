STEPUP

STEPUP Image Analysis (SIA) is a tool for easy photometric analysis that I devloped in the summer of 2017 in Python. Though it was originally intended specficially for STEPUP's use to process photometric data of light curves, it has since been expanded for general astronomical use and is being developed for implementation at the Allgheny Observatory. Check out SIA's GitHub to download and view documentation and user instructions!
